# 🌱 SustainX – Smart Waste Management System

SustainX is an IoT + MERN stack based system that monitors garbage levels in real-time using ESP32 and sensors.

---

## 🚀 Features
- 📡 Real-time bin monitoring
- 🌐 Cloud storage using MongoDB
- 🔔 Alert system when bins are full
- 📊 Admin dashboard
- 📍 Location-based tracking (future scope)

---

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Hardware: ESP32, Sensors
